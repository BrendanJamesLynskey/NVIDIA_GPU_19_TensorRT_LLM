# TensorRT-LLM — NVIDIA's Optimised Inference Engine

The library NVIDIA uses to publish their headline LLM throughput numbers — engine builder, in-flight batching, paged KV-cache, FP8/FP4 quantization, speculative decoding (draft, Medusa, EAGLE), tensor/pipeline/expert parallel, and the Triton Inference Server / NIM serving paths. Honest comparison with vLLM and an interactive engine-config helper.

**Live site:** https://brendanjameslynskey.github.io/NVIDIA_GPU_19_TensorRT_LLM/

Part of the [NVIDIA GPU Architectures series](https://github.com/BrendanJamesLynskey/LLMs#nvidia-gpu-architectures).
